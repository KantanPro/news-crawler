指定されたニュースソースから自動的に記事を取得し、WordPressサイトに投稿として追加するプラグイン

== Description ==

News Crawlerは、設定したニュースソース（RSSフィードやニュースサイト）から自動的に記事を取得し、キーワードにマッチした記事を1つの投稿にまとめて作成するWordPressプラグインです。

**主な機能**

* 複数のニュースソース（RSSフィード、HTMLページ）に対応
* キーワードベースの記事フィルタリング
* 重複記事の自動スキップ
* 品質スコアによる記事評価
* 複数記事を1つの投稿にまとめる機能
* 管理画面での設定と手動実行
* 統計情報の表示

**対応ニュースソース**

* RSSフィード
* ニュースサイトのHTMLページ
* 主要メディアサイト（NHK、朝日新聞、読売新聞など）

**品質管理機能**

* タイトル長のチェック
* 本文長のチェック
* 画像の有無
* 日付情報の有無
* ソース情報の有無

== Installation ==

1. プラグインファイルを `/wp-content/plugins/news-crawler/` ディレクトリにアップロードします
2. WordPressの管理画面でプラグインを有効化します
3. 設定 > News Crawler から設定を行います
4. ニュースソースとキーワードを設定します
5. 「投稿を作成」ボタンで手動実行します

== Frequently Asked Questions ==

= どのようなニュースソースに対応していますか？ =

RSSフィードとHTMLページの両方に対応しています。主要なニュースサイトやブログサイトのURLを指定できます。

= 自動投稿は行われますか？ =

現在のバージョンでは自動投稿機能は廃止されており、管理画面からの手動実行のみとなっています。

= 重複記事はどのように処理されますか？ =

タイトルの類似性とURLの重複チェックにより、重複記事は自動的にスキップされます。

= 品質スコアはどのように計算されますか？ =

タイトル長、本文長、画像の有無、日付情報、ソース情報などを総合的に評価して0.0〜1.0のスコアを算出します。
